# Abstract Algebra · Interactive

**Groups, rings, fields — the algebraic structures underlying all of mathematics**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 4 · Graduate

## What You'll Learn

- Groups: subgroups, homomorphisms, quotient groups
- Sylow theorems and group actions
- Rings: ideals, quotient rings, polynomial rings
- PIDs, UFDs, and Euclidean domains
- Fields: extensions, splitting fields, Galois theory
- Modules (intro)

## References

- David S. Dummit & Richard M. Foote, *Abstract Algebra* (3rd ed., Wiley)
- Thomas W. Hungerford, *Algebra* (GTM 73, Springer)
- Michael Artin, *Algebra* (2nd ed., Pearson)
- Serge Lang, *Algebra* (GTM 211, Springer)
