## Features

- **Interactive visualizations** — Cayley tables, subgroup lattices, permutation animators, Galois correspondence diagrams, all on Canvas 2D
- **KaTeX math rendering** — rigorous notation throughout
- **Exercises with hints & solutions** — from computational to proof-based
- **Progress tracking** — saved locally in your browser
- **Zero dependencies** — just open the link and learn
